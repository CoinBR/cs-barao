Versions

Valve still can't move all players to CS:GO

Currently, Counter-Strike has 3 popular versions, each version with its own your fan base.

Counter-Strike 1.6

The last version of the original game. It still has a huge fan base, since it features with perfection all the aspects that made Counter-Strike legendary such as gameplay, physics and the ability the shine by your own skill or exceling in teamwork.

Counter-Strike: Source

Valve has tried to bring a more realistic approach to the game. Despite it features amazing graphics, Counter-Strike: Source changed the way core elements of the game works, like weapon recoil, wall resistance to bullets, the weight of grenades etc. In the end most of the counter-strike 1.6 players didn’t moved to Counter-Strike: Source.

Counter-Strike: Global Offensive

Recently Valve has released Counter-Strike: Global Offensive. A try to join both the Counter-Strike: Source and Counter-Strike 1.6 players in a new version of Counter-Strike. They tried to create a game that balances the core aspects 1.6 that makes the game enjoyable and competitive with some realistic aspects and graphics improvement started in Counter-Strike: Source. The beta versions of Counter-Strike: Source were highly criticised by professional gamers and fans of both earlier versions of Counter-Strike, specially the 1.6 players. But Valve has been changing the game, and making it remarks the core aspects that have made Counter-Strike so successful.
